Black-Scholes Options Pricer

This project is a simple yet functional implementation of the Black-Scholes model, a widely used formula for pricing European call and put options. Users can input parameters such as stock price, strike price, volatility, time to maturity, and risk-free interest rate to calculate the theoretical option price.

I built this project because of my interest in finance and it allowed me to explore how options are priced while applying mathematical concepts. Along the way, I gained experience in web development/coding (HTML, CSS, Python), plus a deeper understanding and intuition on options prices.

I am currently expanding the project to fetch real world stock prices using an API and to incorporate an American option model.
